##  Project Overview

This project is a Digital Literacy Portfolio created as part of the Digital Literacy course. The objective of this project is to understand and apply essential digital skills such as online safety, professional communication, digital presence, and awareness of cyber threats.

The project consists of five tasks, each focusing on different aspects of digital literacy.
